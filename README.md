# Wordle Infinity

Play Wordle - Infinity! Enjoy endless Wordle puzzles and sharpen your word skills with this infinite version of the popular game.

# Tech Stack
- React
- Tailwind CSS
- Framer Motion
- Vite

# Setup

## Step 1
Install the dependencies

```bash
$ npm install
```

## Step 2
Run the development server

```bash
$ npm run dev
```
